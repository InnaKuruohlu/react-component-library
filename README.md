# React Component Library

A modular React component library with Storybook showcasing UI components such as Input, Toast, Button, and Sidebar Menu.

You can view the live Storybook here:  https://innakuruohlu.github.io/react-component-library/?path=/story/input--default


***

Setup Instructions:

-  Clone the repository:
   ```bash
   git clone https://github.com/InnaKuruohlu/react-component-library.git
   cd react-component-library
- Install dependencies: npm install
- Run Storybook locally: npm run storybook


****

Components Overview

1. Input

- Multi-type input (text, password)

- Password visibility toggle

- Clearable input
